# 我的聊天系统 (My Chat System)

欢迎来到我的聊天系统项目！这是一个简单、易用的在线聊天系统，允许用户实时交流。

## 功能特点

- **实时消息传递**：用户可以即时发送和接收消息。
- **用户在线状态**：显示用户在线或离线状态。
- **历史聊天记录**：查看与任一用户的聊天记录。

## 安装

要在本地安装并运行此聊天系统，请按照以下步骤操作：

1. 克隆仓库到本地机器
git clone https://github.com/yourusername/my-chat-system.git
2. 进入项目目录
cd my-chat-system
3. 安装依赖
npm install
4. 启动应用程序
npm start

## 使用方法

- 访问 `http://localhost:3000` 来查看聊天系统界面。
- 输入用户名并点击“登录”开始聊天。
- 在聊天窗口输入消息并发送。

## 技术栈

- 前端：React, Redux, WebSocket
- 后端：Node.js, Express, MongoDB
- 其他：Webpack, Babel, ESLint
